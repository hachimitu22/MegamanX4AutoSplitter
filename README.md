# MegamanX4AutoSplitter

## 導入方法

1. Releaseから`MegamanX4.asl`をダウンロードする。
2. LiveSplitの`Edit Layout`から`Control > Scriptable Auto Splitter`を追加する。
3. `Script Path`に`MegamanX4.asl`を指定する。
